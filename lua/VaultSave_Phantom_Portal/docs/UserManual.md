# 幻影传送门 (VaultSave: Phantom Portal) - 玩家使用手册

## 功能简介
本模组允许玩家在游戏《生化危机 9》(或相关 RE 引擎游戏) 中保存当前位置与朝向，并随时传送回该位置。
主要用于：
- 跑图练习
- 避免卡死
- 快速回溯

## 核心功能
1. **多槽位存档**：支持多达 10 个存档槽位，每个槽位独立保存位置与备注。
2. **即时传送**：一键保存/读取，无需重新加载游戏存档。
3. **穿墙防卡**：传送过程短暂穿墙，防止因碰撞体积导致的传送失败。
4. **多语言支持**：跟随系统语言，支持简/繁/英/日/韩。

## 操作指南

### 快捷键
- **F5** (默认)：保存当前位置到选中槽位。
- **F9** (默认)：传送回选中槽位的位置。
- **Insert** (REFramework 默认)：打开/关闭模组菜单。

### 菜单设置
在 REFramework 菜单 -> Script Generated UI -> VaultSave: Phantom Travel 中：
1. **Settings**:
   - **Language**: 切换界面语言。
   - **Binding**: 点击按钮后按下键盘按键或手柄组合键更改 F5/F9 绑定。
2. **Slot Selection**:
   - 勾选槽位以激活当前操作目标。
   - 点击 **Add Slot (+)** 增加新槽位。
   - 点击 **-** 删除不需要的槽位。
   - 在输入框中编辑槽位备注。
3. **Manual Save/Load**:
   - 点击按钮手动执行保存或读取。

## 使用限制与注意事项
1. **禁止跨图传送**：本模组仅记录坐标，**不记录场景/关卡信息**。在 A 场景保存，去 B 场景读取，会导致你传送到 B 场景的该坐标处（可能是虚空或墙里）。**请务必在同一张地图内使用！**
2. **第一人称视角**：传送后视角朝向可能需要手动微调。
3. **手动跳跃**：传送后如果发现角色浮空或卡住，请尝试移动或跳跃以重置物理状态。
4. **存档兼容性**：模组配置保存在 `reframework/data/VaultSavePhantomTravel/data.json`，删除该文件可重置所有设置。

## 常见问题
- **Q: 按 F9 没反应？**
  - A: 检查左上角是否有提示消息。如果显示 "Slot Empty"，请先按 F5 保存。确保 REFramework 菜单未拦截输入。
- **Q: 传送后掉出地图？**
  - A: 可能是跨图传送导致，或者保存位置过于靠近墙壁外部。
